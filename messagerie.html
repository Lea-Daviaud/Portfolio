Application de messagerie en Java

Lien GitHub : https://github.com/Lea-Daviaud/ChatEnLigne.git

Présentation du projet

Dans le cadre de mon BTS SIO option SLAM, j’ai participé à la réalisation d’une application Jave permettant d'envoyer des messages à des groupes d'utilisateurs leur permettant ainsi de communiquer entre eux. Les utilisateurs peuvent également créer des groupes.


Organisation du projet

Le projet a été mené en mode projet avec un groupe de trois développeurs. Nous avions commencé par élaborer un schémas de base de données. Nous n'avions pas de wireframe précis, ni de répartition des tâches écrite ni de planning.

Pour mettre notre travail en commun, nous utilisions GitLab.


Technologies et techniques utilisées

  • Nous travaillions sur Eclipse en Java.
  • Pour les interfaces, nous utilions JFrame.
  • Et Connection pour faire des requêtes à la base de données.

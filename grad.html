Projet Grad - Site vitrine

Lien GitHub : https://github.com/Lea-Daviaud/Grad.git

Présentation du projet

Dans le cadre de mon BTS SIO option SLAM, j’ai participé à la réalisation d’un site web vitrine dédié à l’aménagement extérieur en bois. Le but était de mettre en avant les produits et les services de l'entreprise à travers un site web.


Organisation du projet

Le projet a été mené en mode projet avec un binôme de développeurs. Nous avions une note de cadrage avec le wireframe prédéfini. Pour ce premier projet de groupe, l'organisation était assez informelle : nous n'avions pas de planning, ni de schémas de base de données ou d'architecture et nous échangions oralement au début de chaque séance pour répartir les tâches du jour.


Technologies et techniques utilisées

  • PHP pour le traitement des formulaires, la logique serveur et la gestion des sessions (session_start()).
  • PDO pour les connexions sécurisées à la base de données MySQL, avec prévention des injections SQL.
  • Utilisation de sha256 pour le hachage des mots de passe.
  • Intégration d’une navbar fixe et d’un footer sur chaque page.
  • Ajout de modals pour visualiser les images du site.
  • CSS et Bootstrap pour être responsive.
  • Création de formulaires pour la collecte d’informations avec traitement sécurisé côté serveur.

Amélioration de la compétence professionnelle

Ce projet m’a permis d’approfondir la compétence suivante du référentiel BTS SIO SLAM : “Développer la présence en ligne de l’organisation”.

En effet, grâce à la mise en place d’un site vitrine complet, j’ai appris à :

  • Structurer un site web autour des besoins d’un client.
  • Construire un site web avec une utilisation intuitive.
  • Utiliser des pratiques de développement sécurisées.
